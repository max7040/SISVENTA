# SISVENTA

Sistema básico de ventas (POS) hecho en **Laravel**, con roles **admin** y **vendedor**.
- Admin: gestiona productos/clientes y ve ventas
- Vendedor: registra ventas y genera comprobante PDF

## Tecnologías
- Laravel (PHP 8.2+)
- MySQL
- Blade + Tailwind (Breeze)
- DomPDF (barryvdh/laravel-dompdf)

## Requisitos
- PHP 8.2+
- Composer
- Node.js (20+ recomendado)
- MySQL (o MariaDB)
